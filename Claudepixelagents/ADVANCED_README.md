## 🚀 **Что умеет продвинутый бот:**

### 🎯 **Создание по промптам**
```python
# Пример промпта:
"Создай элегантного бота для итальянского ресторана 'La Bella'. 
Бот должен знать всё о пасте и вине, говорить по-итальянски базовые фрази, 
уметь рекомендовать блюда по предпочтениям гостя."
```

### ⚙️ **Гибкая настройка данных**
- **Google Sheets** - автоматическая синхронизация меню
- **API интеграции** - подключение CRM, систем бронирования
- **Файловые источники** - JSON, CSV, Excel, PDF
- **Множественные источники** - одновременная работа

### 📋 **Умная загрузка контента**
- **Пересылка файлов** прямо в Telegram
- **AI-парсинг** меню и прайс-листов
- **Автоматическая классификация** блюд
- **Извлечение цен** и описаний

### 🌐 **Веб-интеллект**
- **Поиск информации** о ресторане в интернете
- **Автообновление** данных с сайтов
- **Анализ конкурентов** и цен
- **Сбор отзывов** и репутации

### 🧠 **AI-персонализация**
- **4 типа характера:** Эксперт, Консьерж, Дружелюбный, Консультант
- **Настройка стиля:** от формального до неформального
- **Генерация ответов** с учетом личности
- **Эмоциональная окраска** сообщений

### 💰 **Sales-стратегии**
- **Консультативные продажи** - экспертный подход
- **Импульсивные** - создание срочности
- **Премиум** - эксклюзивность и статус
- **Партнерские** - лояльность и повторные продажи

## 🚀 **Быстрый старт:**

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка .env
```bash
TELEGRAM_BOT_TOKEN=your_token_here
GROQ_API_KEY=your_groq_key
GREEN_API_INSTANCE=your_instance
GREEN_API_TOKEN=your_token
```

### 3. Запуск
```bash
python advanced_ai_architect.py
```
